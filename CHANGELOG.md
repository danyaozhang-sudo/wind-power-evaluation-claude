# Changelog

## v2.0 (2026-06-06)

### 新增
- **Word 报告格式固化**：页面布局（A4/边距）、字体层级（黑体H1/仿宋正文）、表格样式（深蓝表头+白字+交替行底色）、七段免责声明模板、江能Logo页眉
- **双风格 PPT 输出**：投行研报风格（Midnight Executive 白底深蓝 12页）+ 路演 Pitch Deck 风格（12页），两种风格通过 `/pptx` 技能生成
- **程序化视觉 QA**：PPTX 解包 XML → 自动检查对比度/配色/Key Insight 颜色（冰蓝字在白底、白字在白底、橙色 Key Insight 三大失败模式）
- **Obsidian 双备份**：所有输出文件（.md/.docx/.pptx）自动同步到 `wiki/topics/evaluations/{项目名}/`
- **报告输出模式选择**：完整版（投资评估）vs 精简版（基金用途）
- **飞书消息预览推送**

### 变更
- 报告结构从 7 章扩展为包含 [必选A-D] 电价分析 + 限电率评估的完整体系
- 子技能调用明确：wind-power-evaluation-claude 处理电价/限电率/财务模型
- `mlt_ratio` 默认行为从 1.0 改为强制校验（机制电量比例 ≠ 100%）

### 修复
- 机制电价参数陷阱：省定基准小时 vs 上网电量百分比法、混淆申报和中标比例、扣费前后混淆、现货季节偏差
