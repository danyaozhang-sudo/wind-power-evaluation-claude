# 风电项目投资评估

按照以下步骤完成风电项目投资评估。

## 第零步：前置准备（每次评估前必须执行）

### 0.1 搜索 Obsidian 知识库

在开始任何搜索之前，先搜索 Obsidian 知识库中是否有该项目的已有资料：

- 在 `entities/` 目录下搜索项目名、省名、投资方
- 在 `concepts/` 目录下搜索该省已有的新能源规划、电网规划文件
- 在 `topics/` 目录下搜索该省已有的电力市场分析

如果有已有的项目页面或参数文件，优先提取项目基本信息。

**知识库读取限制（硬性规定）**：
- 从 Obsidian 知识库**只能提取**项目所在地和装机容量，用于引导后续搜索
- ⚠️ 以下三类信息**禁止**从知识库提取，必须通过独立搜索获取：
  - 利用小时数
  - 电价
  - 投资概算
- 其他信息（接入点、用地面积、风机台数、建设周期、外线距离、地形地貌等）可从知识库提取，作为辅助判断依据

### 0.2 获取省级规划文件

对于新省份（Obsidian 中无该省规划文件时），搜索并摄取两份关键官方文件：

1. **省级最新新能源规划**：
   - 搜索 `{省份} 新能源 发展 规划 2025 2026`
   - 搜索 `{省份} 十四五 十五五 新能源 装机目标`
   - 搜索 `{省份} 风电 光伏 发展 规划`

2. **省级最新电网建设规划**：
   - 搜索 `{省份} 电网 建设 规划 外送 特高压 2025 2026`
   - 搜索 `{省份} 电力 外送 通道 消纳`
   - 搜索 `{省份} 配电网 储能 规划`

提取关键信息（装机目标、外送通道规划、配储要求等），写入 Obsidian：
- `concepts/{省份}新能源规划.md`
- `concepts/{省份}电网规划.md`

如 Obsidian 中已有这两个文件（6个月内的），跳过搜索和摄取。

## 第一步：获取项目基本信息

询问用户以下必要信息（如未提供）：

1. **项目所在地**：精确到省份、城市/地区（例如：内蒙古乌兰察布、甘肃酒泉、江苏盐城等）。这是搜索电力市场数据的关键依据。
2. **项目装机容量**：以MW为单位（例如：50MW、100MW、200MW）。
3. **项目接入方案**（可选）：用户可提供新建线路接入变电站的信息，辅助判断断面阻塞情况。

如用户上传包含项目信息的文件（Excel、Word、PDF等），优先从文件中提取。

### 1.2 利用小时数获取（硬性规定）

利用小时数分**两步**获取：

**第一步：取低值输入财务模型**
- 通过 `web_search` 全方位搜索（不少于10个不同类型的来源），查询分析得到**理论可利用小时数**
- 从 Obsidian 知识库提取**理论可利用小时数**
- 两者取低值 → 直接输入财务模型（保守原则）

**第二步：实际利用小时数校核**
- 通过 `web_search` 全方位搜索（不少于10个不同类型的来源），查询分析得到项目所在地近两年的**实际可利用小时数**（与第一步的搜索不是同一口径）
- 从 Obsidian 知识库提取**实际可利用小时数**
- 两者取低

**第三步：交叉验证限电率**
- 用理论值和实际值计算隐含限电率 = `1 - (实际小时数 / 理论小时数)`
- 与通过限电率估算（见第三步）得到的限电率交叉验证
- 若两者接近 → 数据可信；若差异大 → 需分析原因，取保守值

**第四步：最终输入**
- 理论可利用小时数（取低值）输入财务模型
- 实际利用小时数用于校核限电率

### 1.3 电价获取（硬性规定）

电价获取规定：
- **禁止**从 Obsidian 知识库中可研类报告直接提取（可信度不足）
- 可从知识库中其他类型知识（如市场分析文章、行业数据汇编）提取电价参考值
- 必须通过 `web_search` 多源搜索验证：**机制电价、双边交易价、现货价三类，每类不少于10个不同类型的来源**
- 知识库参考值与搜索值比对，取保守值（较低者）
- 优先采用官方来源（国家能源局、省级发改委、电力交易中心）
- 存在较大差异时取保守值（较低电价）

**消纳友好省（近负荷中心）**：河南、山东、江苏、广东、浙江、福建、湖北、湖南、江西、安徽等 → 限电率通常 2-6%，搜索重点：省内断面、本地消纳增量

**消纳承压省（外送依赖型）**：甘肃、新疆、宁夏、内蒙古、青海、陕西、山西、河北等 → 限电率通常 8-20%+，搜索重点：**外送特高压进展、就近消纳产业转移、大基地配套**

## 第三步：确定关键参数

### 限电率估算（双轨并行）

**轨道A——项目所在地评估限电率**：
结合接入方案、断面阻塞、消纳能力和省内区域位置：
- **基础三档**：
  - ① 消纳能力≥年发电量1.2倍、无断面阻塞 → 2%-3%
  - ② 消纳能力为1-1.2倍、轻微断面阻塞 → 4%-6%
  - ③ 消纳能力<1倍、严重断面阻塞 → 15%-20%
- **省内区域调整系数**：
  - 同省内，项目位于**消纳较好区域**（近负荷/近外送汇集站/省内南部/东部）→ 在基础档位下限取值
  - 项目位于**消纳较差区域**（远负荷/近电源集中区/省内西部/北部）→ 在基础档位上限取值
  - 调整幅度一般在基础值的 ±15-30%
- **示例**：
  - 甘肃酒泉：全省消纳③（15-20%），酒泉属河西电源集中区取上限 → 18-20%
  - 甘肃陇南：全省消纳③，陇南属河东较好区取下限，且可降一档至② → 10-15%

**轨道B——全省同类型电源平均限电率**（新增）：
- 搜索 `{省份} 风电 全省 平均 限电率 {近两年}`
- 搜索 `全国新能源消纳监测预警中心 {省份} 风电 利用小时数 限电率`
- 搜索 `{省份} 风电 弃风率 全省均值`
- 搜索 `{省份} 风电 限电率 {近两年}年`
- 累计不少于10个不同类型来源交叉验证，取保守值（**较高限电率**）

**最终限电率取值（强制校验）**：
> ⚠️ 将以下三者取**最高值**进入财务模型：
> 1. 轨道A（项目所在地评估）限电率
> 2. 轨道B（全省均值）限电率
> 3. 第2步隐含限电率 = `1 - (实际小时数 / 理论小时数)`

- 以**轨道B（全省均值）**为主要参考基准
- 轨道A作为校核：若估算值显著低于轨道B，可适当下调，但幅度不超过30%
- 若轨道A估算值高于轨道B，取轨道A
- 追加与第1.2步隐含限电率的比对，取三者中**最高值**（保守原则）
- 所有取值均注明依据和来源

### 电价策略

**情况一**（不参与机制电价竞价）：
- 年度中长期交易 70%，按当地中长协均价
- 现货交易 30%，按当地现货均价
- 绿电交易 100%覆盖（与中长协/现货叠加），按绿电均价
- 扣除两个细则考核（0.015元/kWh）和市场费用分摊（结算电费×2%）

**情况二**（参与机制电价竞价）：
- 电量全部竞价，按当地同类项目竞价平均成交价
- 同样扣除两个细则考核和市场费用分摊

### 运维费与保险费

- 运维费：1-5年 0.02元/W·年，6-10年 0.06元/W·年，11-20年 0.08元/W·年
- 保险费：当年项目净值×0.2%（净值=初始投资-累计折旧，折旧按20年直线法，残值3%）

### 融资假设

- **假设1**：融资杠杆率100%（全额融资），融资18年，利率4%，等额本金
- **假设2**：融资杠杆率80%（融资80%，资本金20%），融资18年，利率4%，等额本金

### 增值税抵扣规则

每年增值税销项税可抵扣金额为当年付息金额的13%，可抵扣金额可结转下一年使用，直至融资偿还完毕。

### 任务1（投资边界）：100%融资，最小DSCR≥1.2

**约束条件**：
- 偿债备付率 = 可用于还款的净现金流量 / 当年应还本付息金额
- 经营期内最小DSCR ≥ 1.2

**输出指标**：
- 最高单瓦投资金额、度电成本、全投资IRR、年均净现金流量、年均净利润

**每年净利润表（含公式）**：

| 公式 | 说明 |
|------|------|
| `营业收入 = 年发电量 × 有效电价` | 年发电量 = 装机容量 × 1000 × 理论可利用小时数 × (1-限电率) |
| `运维费 = 装机容量 × 单位运维费率` | 分段：1-5年0.02元/W·年，6-10年0.06元/W·年，11-20年0.08元/W·年 |
| `保险费 = 项目净值 × 0.2%` | 净值 = 初始投资 - 累计折旧 |
| `折旧 = 总投资 × (1-残值率) / 20年` | 直线法，残值率3% |
| `利息 = 期初剩余本金 × 利率` | 等额本金还款 |
| `增值税销项 = 营业收入 × 13% / 1.13` | |
| `增值税可抵扣 = 当年利息 × 13%` | 可结转下一年，直至融资偿还完毕 |
| `增值税及附加 = max(0, 增值税销项 - 累计可抵扣) × 12%` | |
| `所得税 = max(0, (营业收入-运维费-保险费-折旧-利息-增值税及附加) × 25%` | |
| `净利润 = 营业收入 - 运维费 - 保险费 - 折旧 - 利息 - 增值税及附加 - 所得税` | |

**每年净现金流量表（含公式）**：

| 公式 | 说明 |
|------|------|
| `EBITDA = 营业收入 - 运维费 - 保险费` | |
| `可用于还款 = EBITDA - 增值税及附加 - 所得税` | |
| `当年应还本金 = 期初本金 / 18年` | 等额本金 |
| `当年应还利息 = 期初剩余本金 × 利率` | |
| `当年应还本息 = 当年应还本金 + 当年应还利息` | |
| `偿债备付率 = 可用于还款 / 当年应还本息` | |
| `净现金流量 = EBITDA - 增值税及附加 - 所得税 - 当年利息 - 当年偿还本金` | |

### 综合投资决策建议

1. **推荐单瓦投资阈值**：以同时满足DSCR和收益率要求的最低单瓦投资作为决策依据
2. **投资边界**：100%融资、DSCR≥1.2下的最高可接受单瓦投资，是项目融资的保底门槛
3. **出售边界**：80%融资下IRR≥6%且资本金IRR≥8%，是项目可行出售的价值上限
4. **风险提示**：限电率波动风险、电价下行风险、建设成本超支风险
5. **项目所在地电力市场综合评价**

## 注意事项

1. 所有市场数据（电价、限电率、利用小时数）必须通过 `web_search` 实际搜索获取，每类不少于10个不同类型来源
2. 知识库仅用于提取项目所在地、装机容量及其他辅助信息；电价、利用小时数、投资概算三类信息必须通过搜索独立获取
3. 存在不确定性时采用保守假设（电价取低、限电率取高、利用小时取低）
4. 所有引用数据需注明来源
5. 财务模型使用 `scripts/financial_model.py` 执行
6. 每次评估必须完成全部2个任务（投资边界、出售边界）的计算，并附每年净利润表和每年净现金流量表
